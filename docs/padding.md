# Padding

**Format:**
```bash
.p-{size}
.p{side}-{size}
```

**CSS:** `padding, padding-top, padding-right, padding-bottom, padding-left`

**Defaults:**
```bash
padding: [0, 0.125, 0.25, 0.5, 0.75, 1, 1.25, 1.5, 1.75, 2, 2.25, 2.5, 2.75, 3, 4, 5, 6]
```

**Output:**
```css
.p-2{
    padding: 2rem;
}
.pr-1{
    padding-right: 1rem;
}
.pl-0.25{
    padding-left: 0.25rem;
}
```
