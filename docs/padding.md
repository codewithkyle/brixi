# Padding

**Format:**

```bash
.p-{size}
.p{side}-{size}
```

**CSS:** `padding`, `padding-top`, `padding-right`, `padding-bottom`, `padding-left`

**Defaults:**

```bash
padding: [0, 0.125, 0.25, 0.5, 0.75, 1, 1.25, 1.5, 1.75, 2, 2.25, 2.5, 2.75, 3, 4, 5, 6]
```

**Examples:**

```html
<div class="p-1">
    <!-- ...snip... -->
</div>

<div class="px-1 py-1">
    <!-- ...snip... -->
</div>

<div class="pl-1 pt-1 pr-1 pb-1">
    <!-- ...snip... -->
</div>
```
