# Background Colors

**Format:**

```bash
.bg-{name}
.bg-{name}-{shade}
.bg-{name}/{opacity}
.bg-{name}-{shade}/{opacity}
```

**CSS:** `background-color`

**Defaults:** see [Colors](https://brixi.dev/colors)

**Examples:**

```html
<div class="bg-white">
    <div class="bg-grey-50"></div>
    <div class="bg-primary-500/15"></div>
</div>
```
