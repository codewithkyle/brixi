# Background Colors

**Format:**
```bash
.bg-{name}
.bg-{name}-{shade}
```

**CSS:** `background-color`

**Defaults:** see [Colors](https://github.com/codewithkyle/brixi/wiki/Colors)

**Output:**
```css
.bg-white{
    background-color: var(--white);
}
.bg-black{
    background-color: var(--black);
}
.bg-primary-100{
    background-color: var(--primary-100);
}
```
