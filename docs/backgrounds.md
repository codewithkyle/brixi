# Background Colors

**Format:**

```bash
.bg-{name}
.bg-{name}-{shade}
```

**CSS:** `background-color`

**Defaults:** see [Colors](https://github.com/codewithkyle/brixi/wiki/Colors)

**Examples:**

```html
<div class="bg-white">
    <div class="bg-grey-50"></div>
    <div class="bg-primary-500"></div>
</div>
```
