## Margins

**Format:** `.m{side}-{size}`

**CSS:** `margin, margin-top, margin-right, margin-bottom, margin-left`

**Defaults:**
```bash
margins: [0, 0.125, 0.25, 0.5, 0.75, 1, 1.25, 1.5, 1.75, 2, 2.25, 2.5, 2.75, 3, 4, 5, 6]
```

**Output:**
```css
.m-2{
    margin: 2rem;
}
.mr-1{
    margin-right: 1rem;
}
.ml-0.25{
    margin-left: 0.25rem;
}
```
