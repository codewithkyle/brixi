# Margins

**Format:**

```bash
.m-{size}
.m{side}-{size}
```

**CSS:** `margin`, `margin-top`, `margin-right`, `margin-bottom`, `margin-left`

**Defaults:**

```bash
margins: [0, 0.125, 0.25, 0.5, 0.75, 1, 1.25, 1.5, 1.75, 2, 2.25, 2.5, 2.75, 3, 4, 5, 6]
```

**Examples:**

```html
<div class="m-1">
    <!-- ...snip... -->
</div>

<div class="mx-1 my-1">
    <!-- ...snip... -->
</div>

<div class="ml-1 mt-1 mr-1 mb-1">
    <!-- ...snip... -->
</div>
```
