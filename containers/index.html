Custom Width

Format: .w-{name}

CSS: width & max-width

Defaults:

mobile: 411
tablet: 768
laptop: 1024
desktop: 1280
hd: 1920
4k: 3840

Output:

.w-tablet{
    width: 768px;
}
.max-w-4k{
    max-width: 3840px;
}

Fractions

Format: .w-{width}/{total}

CSS: width

Defaults:

columns: [2, 3, 4]

Output:

.w-1/2{
    width: 50%;
}
.w-3/4{
    width: 75%;
}
